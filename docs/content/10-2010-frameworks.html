Framework explosion — 2010

  • Around 2010, a number of JavaScript frameworks emerged:
    • Knockout (July 5, 2010)
    • Backbone (Oct. 13, 2010)
    • AngularJS (Oct. 20, 2010)
    • Ember (Dec. 8, 2011)
  • Each of these attempted to solve the same problem: giving scope to the DOM
  • AngularJS became the de facto standard
