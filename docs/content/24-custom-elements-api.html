Custom Elements API

  • static observedAttributes: [string] — an array of strings to watch for changes to trigger the attributeChangedCallback
  • Provide basic lifecycle methods
    • connectedCallback — fired when the element is attached to the DOM tree
    • disconnectedCallback — fired when the element is removed from the DOM tree
    • adoptedCallback — fired when the element is adopted into a new part of the DOM tree
    • attributeChangedCallback — fired when one of the element's attributes changes if that attribute is included in the element's static observedAttributes
