Shadow DOM — slot

The following code uses an invalid element to represent the shadow DOM

        
<style> .tomato { color: tomato; } </style>

<div>
<p class="tomato">This content is tomato.</p> <!-- Will display in slot -->
<#shadow-root>
    <style> .tomato { color: papayawhip; } </style>
    <p class="tomato">This content is papayawhip.</p>
    <slot></slot> <!-- The light DOM p.tomato will appear in this location -->
</#shadow-root>
</div>

This content is tomato

This content is tomato
